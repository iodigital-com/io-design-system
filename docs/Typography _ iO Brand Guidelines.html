Brand assets
Typography
Our primary font is Manrope – a free, open-source Google font. It's easy to share, download, and use across all platforms, which makes it ideal for Microsoft Office tools like PowerPoint and Word.  
For selected external channels (social media, ads) we use TT Commons Pro.
All treatment, sizing, placement, do’s and dont’s applies to both Manrope and TT Commons Pro.

Typography
Primary font

Our Primary font is Manrope. We use Light and Medium for body text, and Bold and Medium for headlines. So, if you want to be on the safe side, always use Medium. 

Download Manrope

Typography
Secondary font

For our paid ads and social media, we use TT Commons Pro. This is a licensed font and not available to everyone. If you need a licence, reach out to Karin Engdahl at karin.engdahl@iodigital.com.

Typography
Guiding principles


Keep it simple
Don’t feel the need to overdecorate the area. Feel confident in our expression. Negative space is a design element. Don’t be scared to give content the space to breathe.


Create contrast
Create visual hierarchy, especially through typographic contrast. Look for the right moments to be expressive. And feel free to let typography be the hero. When highlighting important subtitles or headers, the DemiBold weight can be used, but do it sparingly.


Mix weights with care
Before adding another weight to a design, always ask yourself: in what way does the design become better by adding this? Does it make my message more clear? Does it make it easier for a user to understand?

Typography
Treatment


Sentence case / Upper case
We primarily use sentence case for all of our copy – headings, preambles and body copy. We only use upper case sparingly to visually seperate a message. For example buttons, overlines and our dynamic tagline.


Alignment
Left-aligned text is always our go-to option when the text is longer than a headline. Use centred text when it suits the design, but avoid using it for body copy. Right-aligned text is considered an exception and is only used for unusual media and layouts.


Font colour
We use our base colours or white text on images and on our accent colour. Use a dark grey over our base colours. We only use our accent colours for short headlines.


Line height and letter spacing
The optimal line height varies depending on the length and style of the text. If you’re uncertain, use 110% for headlines and 150% for body text. Bigger text requires tighter letter spacing, smaller text requires broader letter spacing.

Typography
Sizing

To create a proportional hierarchy – good visual balance and legibility – we recommend using a modular scale to create font sizes relevant for your project. You can easily use an online service such as Typescale.com, to calculate it for you. The recommended scale to use is Major Third.

Typography
Do's and don'ts


Do
Be confident in our message and create typographic expressions that let them stand out.


Do
To create a proportional hierarchy and a good visual balance, we recommend using a modular scale to create font sizes relevant for your project.


Don't
Never use other weight than the ones approved for the primary and fallback font.


Don't
Avoid using to many different styles on your text. Learn more in the chapter Rule of thumb


Don't
Never mix fonts.


Don't
Never use Manrope in paid ads or our social media assets.


Don't
Never use low contrast colours. Avoid using colours other than white, black or our primary blue on images.


Don't
Never use blue text on images where there is already a lot going on.
