Colours

The colour palette is one of our brand’s most powerful and recognisable assets. The most recognisable colour is Energetic Blue. Our four base colours provide a backdrop for much of the brand. In combination with the Energetic Blue, it creates a dynamic and recognisable combination. All four base colours have a toned down version for when we need to unify the overall expression.

This creates a broad yet defined colour palette that’s suitable for a range of different applications.

Whitespace

Combinations

Contrast

Colour balance

Background

Select one of the base colours if the message is complex.


Tone the background colour down when adding more elements.
